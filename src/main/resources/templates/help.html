Description

This application is work in progress. The content is there just as a proof of concept. The ultimate goal is to create a library of sign language signs, where anyone can search for signs, browse, view, download or integrate the content in any way they want.

Search

The main page can be used to search for any kind of sign with free form search. For example you can search for "train". By default, the results are shown as an HTML page. The same results can be retrieved as JSON by adding the format=json query parameter or as Atom / OpenSearch by adding the format=atom query parameter.

Signs

Each sign has a page where you can see all information about that sign. For example, the "train" has its own page. This can again be viewed as JSON.

Swagger UI

A Swagger endpoint is available to browse the available API endpoints and help you get familiar with the kind of information available on this application.
